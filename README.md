### Binary Code
- BCD Code (8421 Code) : Only use 4 bits to represent each digit.
- Excess-3 Code : BCD Code + 0011<sub>(2)</sub>
- Gray Code : Two successive values differ in only one bit. (Used in Rotary Encoder)
